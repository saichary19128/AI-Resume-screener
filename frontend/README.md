## Setup

- Backend:
  - `cd backend`
  - copy `.env.example` to `.env` and fill `MONGO_URI` and `OPENAI_API_KEY`
  - `npm install`
  - `npm run dev`

- Frontend:
  - `cd frontend`
  - `npm install`
  - `npm start`

The frontend expects backend at `http://localhost:5000/api` by default. Set `REACT_APP_API_URL` if different.
